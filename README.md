## âœ¦ Quick Start

### 1. Initialize the Ritual
```bash
npx cine-ritual init my-project
```

### 2. Prepare Your Soul (Media)
Place your `.mp4` video in the `/source` folder.
```bash
npm run process-media
```

### 3. Evoke the Aesthetic
Edit `cineritual.json` to define your brand colors and typography. The engine handles the rest.
