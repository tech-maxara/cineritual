# ✦ CineRitual
> The World's Easiest AI-Powered 3D & Cinematic Website Builder. Create a masterpiece in 5 minutes.

![CineRitual Banner](assets/hero.gif)

**CineRitual** is a high-performance framework designed to transform cold, static websites into mesmerizing, 3D-feeling cinematic rituals. No Three.js complexity. No Spline learning curve. Just stunning results in minutes.

---

## ✦ Why CineRitual?
*   **5-Minute Setup**: Go from raw video to a live cinematic 3D site in record time.
*   **The World's Easiest Logic**: No complex 3D math. If you can scroll, you can animate.
*   **AI-Native Promptbook**: Built-in master prompts for Gemini, Midjourney, and Flux to generate 3D assets instantly.
*   **Ultra-Premium Aesthetics**: One-click glassmorphism, storytelling typography, and lighting engines.

---

## ✦ Quick Start

### 1. Initialize the Ritual
```bash
npx cine-ritual init my-project
```

### 2. Prepare Your Soul (Media)
Place your `.mp4` video in the `/source` folder.
```bash
npm run process-media
```

### 3. Evoke the Aesthetic
Edit `cineritual.json` to define your brand colors and typography. The engine handles the rest.
